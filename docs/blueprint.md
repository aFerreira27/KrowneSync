# **App Name**: Synchromatic

## Core Features:

- User Authentication: Secure user authentication via token system.
- Account Linking: Allow users to securely connect to Salesforce, Salespad, Autoquotes, and website management software using tokens.
- Data Comparison: Automatically identify differences between databases. Uses a tool to recognize whether these differences could impact the overall data integrity.
- Automated Updating: Update databases with correct product information, while respecting platform-specific validation rules.
- PDF Generation: Generate product PDFs using pdfme based on current product data.
- PDF Upload: Upload generated PDFs to the various databases via API calls.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to represent stability and trustworthiness.
- Background color: Light Blue (#E8EAF6) for a clean and professional interface.
- Accent color: Orange (#FF9800) for highlighting important actions and alerts.
- Body font: 'Inter', sans-serif, for a modern and readable text.
- Headline font: 'Space Grotesk', sans-serif, for a computerized, techy, scientific feel.
- Use clear and professional icons to represent different data sources and actions.
- Design a clean and intuitive dashboard layout with clear data visualization.
- Use the provided logo image (a green 'SYNC' inside a green capsule shape) on the landing page.
